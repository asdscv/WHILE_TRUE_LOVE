# 갤러리 사진 폴더

이 폴더에 사진 파일(`.jpg`, `.jpeg`, `.png`, `.webp`)을 넣으면
**자동으로 갤러리에 표시**됩니다. 개수 제한 없음(수십 장 OK).

## 사용법
1. 사진 파일을 이 폴더에 복사합니다.
2. 표시 순서를 정하고 싶으면 파일명 앞에 번호를 붙이세요.
   예) `01.jpg`, `02.jpg`, `03.jpg` …
3. 첫 번째 사진(`01.*`)은 메인(Cover) 배경으로도 사용됩니다.

> 이 폴더가 비어 있는 동안에는 임시 이미지(picsum.photos)가 대신 표시됩니다.
> 실제 사진을 넣으면 임시 이미지는 자동으로 사라집니다.
