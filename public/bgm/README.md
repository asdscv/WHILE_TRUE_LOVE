# 배경음악 폴더

여기에 `mp3` 파일을 넣고 `src/config.js` 의 `bgm.file` 값을 파일명으로 바꾸세요.

```js
// src/config.js
bgm: {
  enabled: true,
  file: 'bgm.mp3', // ← 이 폴더에 넣은 파일 이름
}
```

- 페이지 진입 시 재생됩니다. (브라우저 정책상 소리는 화면을 한 번
  터치/스크롤한 뒤 시작됩니다. 우측 상단 ♪ 버튼으로 켜고 끌 수 있습니다.)
- 곡을 바꾸려면 이 폴더의 파일을 교체하거나 `bgm.file` 값만 바꾸면 됩니다.
- 저작권에 유의하여 사용 가능한 음원을 넣어주세요.
